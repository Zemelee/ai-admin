## 二、规则引擎扩展
- [ ] 用工红线规则：休息日/法定节假日上班、夜班、强制加班、收押金/培训费 → 红色高危
- [ ] 薪酬合规规则：薪资低于同岗 80% 或当地最低工资、发放周期 > 1 月 → 红色
- [ ] 学生准入规则：年龄 < 16 / 一年级在校生 → 阻断
- [ ] 实习时长规则：总时长 < 6 个月预警；变更前置条件（满 1 月 / 满 3 月）扩展校验
- [ ] 自动巡检定时任务（每日全量）：覆盖以上全部规则

## 三、AI 能力升级

- [ ] 实习成绩辅助核算：聚合企业评分 + 考勤完成率 + 日志完整度，AI 给出参考成绩
- [ ] 企业评价初稿生成：基于该企业学生历史数据，AI 批量生成周/月度评价初稿
- [ ] 语音转文字录入（学生日志/周记 Web Speech API + 后端 fallback）
